Модули - это как отдельные приложения между модулями могут быть зависимости, но зависимости циклическими быть не должны, иначе будет ошибка
В папке с модулем для передачи информации создается module-info для импорта и эскпорта

export - для экспорта (отдать)
requires - для импорта (всосать)

Для подключения клиент должен знать адрес серверной машины в сети IP и порт,
на котором программа-сервер запущена и ожидает клиентов

Модель: клиент подключается к серверу и не разрывает связь (открытое соединение или Socket соединение)

Уровень 4 - транспортный уровень (нас интересует)
http протоколы - 7ой уровень

Если данные передаем по tcp протоколу, то передача данных гарантируется в полном объеме
НО накладные расходы - будет вестись учеты, все ли дошло (все важные данные отправляем по нему)

jdp протокол - нет учета передалось ли все (например компьютерные онлайн-игры - нет ничего страшного в том, если часть картинки потеряются)

tcp и jdp протоколы работают только с байтами, поэтому перед отправкой все преобразовываем в байты


Если программа однопоточная, то серверная программа сможет работать только с одним клиентом и с другим не может, пока не разорвет соединение
При работе с io - только однопоточный вариант возможен
nio - есть многопоточный, но позже
